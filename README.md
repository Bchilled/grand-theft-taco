# 🌮 Grand Theft Taco: IRS Collections

**A Whopper Jr Game**

> *"we put the meat in the code"*

A top-down 2D action game where you play as TacoMan — a large taco with skinny arms, dual-wielding weapons, running an underground taco empire while evading the IRS.

## 🎮 Play Now

👉 **[Play the Demo](https://bchilled.github.io/grand-theft-taco)**

## Controls

| Key | Action |
|-----|--------|
| `WASD` | Move |
| `Mouse` | Aim |
| `Left Click` | Shoot |
| `Q` | Switch weapon (Glock / Corn RPG) |
| `E` | Interact (collect stands, grab weapons, talk to homies) |
| `H` | Heal using ingredients |

## Phase 1 Features

- 🌮 TacoMan — big taco body, skinny arms, dual Salsa Glocks
- 🌽 Corn RPG — explosive area damage (start with 5 shots)
- 👮 IRS Agents — chase, shoot, escalate in waves
- 🏪 7 Taco Stands — collect illegal money, raise IRS heat
- 🧑 Homies — give ammo and tips
- 🥩 Ingredients — pick up and press H to heal +25 HP each
- 🗺️ Large city — 7x7 block grid with wide streets
- 📊 Minimap — shows stands, homies, RPG pickups, enemies
